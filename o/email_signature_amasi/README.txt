Email Signature Files

1) gmail_signature.html
   - Table-based HTML signature for Gmail.
   - Open in browser, select the signature area, copy, then paste into Gmail Settings > Signature.

2) outlook_signature.html
   - Table-based HTML signature for Outlook.
   - Use the same copy-paste method in Outlook signature settings.

3) image_only_signature.html
   - Exact image version. Only the full image is clickable.

Important:
- For production email signatures, upload the PNG files in /assets to your website/server and replace image paths like assets/legend_left_panel.png with full HTTPS URLs.
- Replace social media placeholder links with the real Legend Group profile links.
